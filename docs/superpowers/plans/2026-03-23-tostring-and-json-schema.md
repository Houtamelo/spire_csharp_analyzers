## Task 1: ToStringEmitter — Struct Strategies

**Files:**
- Create: `src/Spire.SourceGenerators/Emit/ToStringEmitter.cs`

### Design

`ToStringEmitter` is a static class with a single public method per strategy category:

- `EmitStructToString(SourceBuilder sb, UnionDeclaration union)` — switch on `kind`, format each variant. Used by all 5 struct emitters (Additive, Overlap, BoxedFields, BoxedTuple, UnsafeOverlap).
- `EmitRecordClassToString(SourceBuilder sb, UnionDeclaration union)` — emits a separate `override string ToString()` inside each variant's partial re-declaration.

For struct strategies, field access uses the public property names (e.g., `this.radius`, `this.sideLength`). These are the same properties used by JSON Write and pattern matching — they abstract over the internal slot layout.

For record/class strategies, field access uses `this.FieldName` — the positional parameters (records) or user-defined properties (classes) declared on the variant type.

### Generated code — Struct example

For `Shape` with variants `Circle(double radius)`, `Square(int sideLength)`, `Point()`:

```csharp
public override string ToString()
{
    return this.kind switch
    {
        Kind.Circle => $"Circle({this.radius})",
        Kind.Square => $"Square({this.sideLength})",
        Kind.Point => "Point()",
        _ => "Shape(?)",
    };
}
```

### Generated code — Record/Class example

For record `Shape` with variants `Circle(double Radius)`, `Square(int Side)`:

Emitted as a **separate generated file** (`{TypeName}.ToString.g.cs`):

```csharp
// <auto-generated/>
namespace Shapes
{
    public abstract partial record Shape
    {
        public sealed partial record Circle
        {
            public override string ToString() => $"Circle({this.Radius})";
        }
        public sealed partial record Square
        {
            public override string ToString() => $"Square({this.Side})";
        }
    }
}
```

```csharp
using System.Linq;
using Houtamelo.Spire.SourceGenerators.Model;

namespace Spire.SourceGenerators.Emit;

/// Generates ToString() overrides for discriminated unions.
internal static class ToStringEmitter
{
    /// Emits ToString() for struct strategies (Additive, Overlap, BoxedFields, BoxedTuple, UnsafeOverlap).
    /// Must be called inside the partial struct body.
    public static void EmitStructToString(SourceBuilder sb, UnionDeclaration union)
    {
        sb.AppendLine();
        sb.AppendLine("public override string ToString()");
        sb.OpenBrace();
        sb.AppendLine("return this.kind switch");
        sb.OpenBrace();

        foreach (var variant in union.Variants)
        {
            if (variant.Fields.Length == 0)
            {
                sb.AppendLine($"Kind.{variant.Name} => \"{variant.Name}()\",");
            }
            else
            {
                var interpolation = string.Join(", ",
                    variant.Fields.Select(f => $"{{this.{f.Name}}}"));
                sb.AppendLine($"Kind.{variant.Name} => $\"{variant.Name}({interpolation})\",");
            }
        }

        sb.AppendLine($"_ => \"{union.TypeName}(?)\",");
        sb.CloseBrace(";"); // switch expression
        sb.CloseBrace(); // method
    }

    /// Emits a separate generated file with ToString() overrides for record/class variants.
    public static string EmitRecordClassToString(UnionDeclaration union)
    {
        var sb = new SourceBuilder();
        sb.AppendLine("// <auto-generated/>");
        sb.AppendLine("#nullable enable");
        sb.AppendLine();

        var hasNamespace = !string.IsNullOrEmpty(union.Namespace);
        if (hasNamespace)
        {
            sb.AppendLine($"namespace {union.Namespace}");
            sb.OpenBrace();
        }

        sb.OpenContainingTypes(union.ContainingTypes);

        var typeParams = FormatTypeParams(union.TypeParameters);
        var unionType = union.TypeName + typeParams;
        var accessMod = string.IsNullOrEmpty(union.AccessibilityKeyword) ? "" : union.AccessibilityKeyword + " ";
        var keyword = union.Strategy == EmitStrategy.Record ? "record" : "class";
        sb.AppendLine($"{accessMod}abstract partial {keyword} {unionType}");
        sb.OpenBrace();

        foreach (var variant in union.Variants)
        {
            var variantAccess = string.IsNullOrEmpty(variant.AccessibilityKeyword) ? "" : variant.AccessibilityKeyword + " ";
            sb.AppendLine($"{variantAccess}sealed partial {keyword} {variant.Name}");
            sb.OpenBrace();

            if (variant.Fields.Length == 0)
            {
                sb.AppendLine($"public override string ToString() => \"{variant.Name}()\";");
            }
            else
            {
                var interpolation = string.Join(", ",
                    variant.Fields.Select(f => $"{{this.{f.Name}}}"));
                sb.AppendLine($"public override string ToString() => $\"{variant.Name}({interpolation})\";");
            }

            sb.CloseBrace(); // variant
        }

        sb.CloseBrace(); // union type

        sb.CloseContainingTypes(union.ContainingTypes);
        if (hasNamespace) sb.CloseBrace();

        return sb.ToString();
    }

    private static string FormatTypeParams(EquatableArray<string> typeParameters)
    {
        if (typeParameters.Length == 0) return "";
        return "<" + string.Join(", ", typeParameters) + ">";
    }
}
```

## Task 2: Update Existing Snapshot Tests for ToString

Every existing snapshot `output.cs` must be updated to include the `ToString()` method in the generated output.

### Strategy

**Struct snapshots:** Add `ToString()` switch expression inside the partial struct body — goes after `IsVariant` properties.

**Record/class snapshots:** The `ToString()` lives in a **separate file** (`{TypeName}.ToString.g.cs`). The existing `output.cs` files compare against `GetUnionGeneratedSource()` which filters out `.ToString.g.cs` files. So existing record/class `output.cs` files do NOT need changes.

Wait — `GetUnionGeneratedSource()` currently filters by excluding `.Stj.g.cs` and `.Nsj.g.cs`. It will pick up `.ToString.g.cs` too unless we update the filter. Two options:
1. Update `GetUnionGeneratedSource()` to also exclude `.ToString.g.cs` — then existing record/class snapshots stay unchanged.
2. Don't emit a separate file for record/class ToString — inline it in the main emitter.

**Decision: Update the filter.** This keeps ToString in its own file (clean separation) and avoids touching all record/class snapshot outputs.

## Task 4: JsonSchemaEmitter — Compile-Time Tier (Fallback)

**Files:**
- Create: `src/Spire.SourceGenerators/Emit/JsonSchemaEmitter.cs`

### Design

The compile-time tier builds the JSON Schema string entirely in the source generator using basic C# type → JSON Schema type mapping. This is the fallback path for all NSJ configs and STJ configs below .NET 9.

The emitter produces a **separate generated file** (`{TypeName}.Schema.g.cs`) containing a partial type declaration with the `JsonSchema` lazy property.

### Steps

- [ ] **Step 1: Create `JsonSchemaEmitter.cs`**

Create `src/Spire.SourceGenerators/Emit/JsonSchemaEmitter.cs`:

```csharp
using System.Linq;
using System.Text;
using Houtamelo.Spire.SourceGenerators.Model;

namespace Spire.SourceGenerators.Emit;

/// Generates a JSON Schema (oneOf + discriminator) for discriminated unions.
/// Two tiers: compile-time string literal (fallback) and .NET 9+ JsonSchemaExporter (STJ only).
internal static class JsonSchemaEmitter
{
    /// Emits a partial type declaration with a Lazy<string> JsonSchema property.
    /// Uses #if NET9_0_OR_GREATER to switch between tiers when STJ is configured.
    public static string Emit(UnionDeclaration union, bool hasStj)
    {
        var sb = new SourceBuilder();
        sb.AppendLine("// <auto-generated/>");
        sb.AppendLine("#nullable enable");
        sb.AppendLine();

        var hasNamespace = !string.IsNullOrEmpty(union.Namespace);
        if (hasNamespace)
        {
            sb.AppendLine($"namespace {union.Namespace}");
            sb.OpenBrace();
        }

        sb.OpenContainingTypes(union.ContainingTypes);

        var typeParams = FormatTypeParams(union.TypeParameters);
        var unionType = union.TypeName + typeParams;
        var accessMod = string.IsNullOrEmpty(union.AccessibilityKeyword) ? "" : union.AccessibilityKeyword + " ";
        var isRecordOrClass = union.Strategy == EmitStrategy.Record || union.Strategy == EmitStrategy.Class;
        var readonlyMod = (!isRecordOrClass && union.IsReadonly) ? "readonly " : "";
        var refMod = (!isRecordOrClass && union.IsRefStruct) ? "ref " : "";
        var abstractMod = isRecordOrClass ? "abstract " : "";

        sb.AppendLine($"{accessMod}{abstractMod}{readonlyMod}{refMod}partial {union.DeclarationKeyword} {unionType}");
        sb.OpenBrace();

        // Lazy<string> property
        sb.AppendLine("public static global::System.Lazy<string> JsonSchema { get; } = new(BuildJsonSchema);");
        sb.AppendLine();
        sb.AppendLine("private static string BuildJsonSchema()");
        sb.OpenBrace();

        if (hasStj)
        {
            sb.AppendLine("#if NET9_0_OR_GREATER");
            EmitStjRuntimeTier(sb, union, unionType);
            sb.AppendLine("#else");
            EmitCompileTimeTier(sb, union);
            sb.AppendLine("#endif");
        }
        else
        {
            EmitCompileTimeTier(sb, union);
        }

        sb.CloseBrace(); // BuildJsonSchema

        sb.CloseBrace(); // type

        sb.CloseContainingTypes(union.ContainingTypes);
        if (hasNamespace) sb.CloseBrace();

        return sb.ToString();
    }

    private static void EmitCompileTimeTier(SourceBuilder sb, UnionDeclaration union)
    {
        var schema = BuildCompileTimeSchema(union);
        // Escape the schema for a verbatim string literal
        var escaped = schema.Replace("\"", "\"\"");
        sb.AppendLine($"return @\"{escaped}\";");
    }

    private static void EmitStjRuntimeTier(SourceBuilder sb, UnionDeclaration union, string unionType)
    {
        // Build the schema at runtime using JsonSchemaExporter.
        // For each variant, get the field type schemas from STJ, wrap in oneOf + discriminator.
        sb.AppendLine("var options = global::System.Text.Json.JsonSerializerOptions.Default;");
        sb.AppendLine("var sb = new global::System.Text.StringBuilder();");
        sb.AppendLine("sb.Append(\"{\\\"oneOf\\\":[\");");

        for (int vi = 0; vi < union.Variants.Length; vi++)
        {
            var variant = union.Variants[vi];
            var jsonVariantName = variant.JsonName ?? variant.Name;

            if (vi > 0) sb.AppendLine("sb.Append(\",\");");
            sb.AppendLine($"sb.Append(\"{{\\\"type\\\":\\\"object\\\",\\\"properties\\\":{{\\\"" +
                $"{union.JsonDiscriminator}\\\":{{\\\"const\\\":\\\"{jsonVariantName}\\\"}}\");");

            foreach (var field in variant.Fields)
            {
                var jsonFieldName = field.JsonName ?? field.Name;
                sb.AppendLine($"sb.Append(\",\\\"{jsonFieldName}\\\":\");");
                sb.AppendLine($"sb.Append(global::System.Text.Json.Schema.JsonSchemaExporter.GetJsonSchemaAsNode(options, typeof({field.TypeFullName})).ToJsonString());");
            }

            // Close properties, add required array
            sb.AppendLine($"sb.Append(\"}},\\\"required\\\":[\\\"{union.JsonDiscriminator}\\\"\");");
            foreach (var field in variant.Fields)
            {
                var jsonFieldName = field.JsonName ?? field.Name;
                sb.AppendLine($"sb.Append(\",\\\"{jsonFieldName}\\\"\");");
            }
            sb.AppendLine("sb.Append(\"]}\");");
        }

        sb.AppendLine($"sb.Append(\"],\\\"discriminator\\\":{{\\\"propertyName\\\":\\\"{union.JsonDiscriminator}\\\"}}}}\");");
        sb.AppendLine("return sb.ToString();");
    }

    /// Builds the complete JSON Schema string at compile time using basic type mapping.
    internal static string BuildCompileTimeSchema(UnionDeclaration union)
    {
        var sb = new StringBuilder();
        sb.Append("{\"oneOf\":[");

        for (int vi = 0; vi < union.Variants.Length; vi++)
        {
            var variant = union.Variants[vi];
            var jsonVariantName = variant.JsonName ?? variant.Name;

            if (vi > 0) sb.Append(",");
            sb.Append("{\"type\":\"object\",\"properties\":{");
            sb.Append($"\"{union.JsonDiscriminator}\":{{\"const\":\"{jsonVariantName}\"}}");

            foreach (var field in variant.Fields)
            {
                var jsonFieldName = field.JsonName ?? field.Name;
                var jsonType = MapTypeToJsonSchema(field.TypeFullName);
                sb.Append($",\"{jsonFieldName}\":{jsonType}");
            }

            sb.Append("},\"required\":[");
            sb.Append($"\"{union.JsonDiscriminator}\"");
            foreach (var field in variant.Fields)
            {
                var jsonFieldName = field.JsonName ?? field.Name;
                sb.Append($",\"{jsonFieldName}\"");
            }
            sb.Append("]}");
        }

        sb.Append($"],\"discriminator\":{{\"propertyName\":\"{union.JsonDiscriminator}\"}}}}");
        return sb.ToString();
    }

    /// Maps a C# type full name to a JSON Schema type fragment.
    internal static string MapTypeToJsonSchema(string typeFullName)
    {
        // Strip nullable wrapper (int? / Nullable<T>)
        var type = typeFullName;
        if (type.EndsWith("?"))
            type = type.Substring(0, type.Length - 1);
        else if (type.StartsWith("System.Nullable<") && type.EndsWith(">"))
            type = type.Substring("System.Nullable<".Length, type.Length - "System.Nullable<".Length - 1);

        return type switch
        {
            "int" or "System.Int32"
                or "long" or "System.Int64"
                or "short" or "System.Int16"
                or "byte" or "System.Byte"
                or "sbyte" or "System.SByte"
                or "ushort" or "System.UInt16"
                or "uint" or "System.UInt32"
                or "ulong" or "System.UInt64"
                => "{\"type\":\"integer\"}",

            "float" or "System.Single"
                or "double" or "System.Double"
                or "decimal" or "System.Decimal"
                => "{\"type\":\"number\"}",

            "string" or "System.String"
                or "char" or "System.Char"
                => "{\"type\":\"string\"}",

            "bool" or "System.Boolean"
                => "{\"type\":\"boolean\"}",

            _ when type.EndsWith("[]")
                or type.StartsWith("System.Collections.Generic.List<")
                or type.StartsWith("System.Collections.Generic.IList<")
                or type.StartsWith("System.Collections.Generic.IEnumerable<")
                => "{\"type\":\"array\"}",

            _ => "{}",
        };
    }

    private static string FormatTypeParams(EquatableArray<string> typeParameters)
    {
        if (typeParameters.Length == 0) return "";
        return "<" + string.Join(", ", typeParameters) + ">";
    }
}
```

```csharp
        var expectedSource = File.ReadAllText(outputPath);

        var result = GeneratorTestHelper.RunGenerator(
            inputSource, out var compilation, out var diagnostics);

        GeneratorTestHelper.AssertNoGeneratorDiagnostics(diagnostics);

        var actualSource = GeneratorTestHelper.GetGeneratedSource(result, ".Schema.g.cs");
        Assert.NotNull(actualSource);

        var actualTree = CSharpSyntaxTree.ParseText(actualSource!);
        var expectedTree = CSharpSyntaxTree.ParseText(expectedSource);

        if (!actualTree.GetRoot().IsEquivalentTo(expectedTree.GetRoot()))
        {
            throw new XunitException(
                $"Snapshot mismatch for case '{casePath}'.\n\n" +
                $"=== EXPECTED ===\n{expectedSource}\n\n" +
                $"=== ACTUAL ===\n{actualSource}");
        }
    }
}

[AttributeUsage(AttributeTargets.Method)]
public sealed class JsonSchemaCaseDiscoveryAttribute : DataAttribute
{
    public override IEnumerable<object[]> GetData(MethodInfo testMethod)
    {
        var casesDir = Path.Combine(AppContext.BaseDirectory, "cases", "discriminated_union", "json_schema");
        if (!Directory.Exists(casesDir))
            yield break;

        foreach (var dir in Directory.GetDirectories(casesDir, "*", SearchOption.AllDirectories))
        {
            if (Directory.GetDirectories(dir).Length > 0) continue;

            var inputPath = Path.Combine(dir, "input.cs");
            var outputPath = Path.Combine(dir, "output.cs");

            if (!File.Exists(inputPath) || !File.Exists(outputPath))
                throw new InvalidOperationException(
                    $"Test case directory '{dir}' must contain both input.cs and output.cs");

            yield return new object[] { Path.GetRelativePath(casesDir, dir) };
        }
    }
}
```
