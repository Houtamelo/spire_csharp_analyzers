# Houtamelo.Spire.PatternAnalysis

This package contains the exhaustiveness analysis engine for Spire's discriminated union pattern checking.

**Do not depend on this package directly.** Use the meta-package instead:

```shell
dotnet add package Houtamelo.Spire
```

`Houtamelo.Spire` includes this package along with the core attributes, analyzers, code fixes, and source generators.
